# QBoard

Example of usage rom-rb and dry-rb with Rails.

## Installation

Setup your database and run migrations:

```ruby
$ rails db:migrate
<= db:migrate executed
```

## Usage

Run server:

```ruby
$ rails s
```

## Usage

Open http://localhost:3000 in your browser

## App deployed to Heroku

https://qaboardapp.herokuapp.com/
